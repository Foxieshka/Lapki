## Авторизация и пользователи

Для доступа к защищенным эндпоинтам передавайте заголовок: 
`Authorization: Bearer <your_access_token>`

| Метод | Эндпоинт | Права | Описание |
| :--- | :--- | :--- | :--- |
| `POST` | `auth/register/` | `AllowAny` | Регистрация. Возвращает данные юзера + JWT-токены |
| `POST` | `auth/login/` | `AllowAny` | Вход (кастомный JWT-сериализатор) |
| `POST` | `auth/logout/` | `IsAuthenticated` | Выход (добавление `refresh` токена в черный список) |
| `PUT/PATCH`| `auth/profile/` | `IsAuthenticated` | Обновление профиля текущего пользователя |
| `POST` | `token/` | `AllowAny` | Стандартное получение JWT-токенов (`Pair`) |
| `POST` | `token/refresh/` | `AllowAny` | Обновление истекшего `access` токена через `refresh` |
| `POST` | `token/verify/` | `AllowAny` | Проверка валидности текущего токена |

## Категории и Животные

### Получить список категорий
* **URL:** `categories/`
* **Метод:** `GET`
* **Права:** `AllowAny`
* **Ответ (`200 OK`):** Список объектов категорий.

### Получить список животных
* **URL:** `animals/`
* **Метод:** `GET`
* **Права:** `AllowAny`
* **Ответ (`200 OK`):** Список объектов животных.

## Товары (`products/`)

Стандартный CRUD-набор эндпоинтов со следующей логикой прав доступа:
* **Просмотр (`GET`):** Доступно всем.
* **Создание (`POST`):** Авторизованные пользователи (автоматически привязывается текущий юзер).
* **Изменение/Удаление (`PUT/PATCH/DELETE`):** Только автор товара (`IsAuthorOrReadOnly`).

### 1. Получить список товаров (с фильтрацией и пагинацией)
* **URL:** `products/`
* **Метод:** `GET`
* **Параметры пагинации (По умолчанию 9 элементов):** `page`, `page_size`.
* **Параметры фильтрации (Query):**
  * `animal` — Фильтр по ID животного.
  * `category` — Фильтр по ID категории.
  * `animal_size` — Размер животного.
  * `in_stock` — Наличие (true/false).
* **Параметры поиска (`search`):** по полям `title` и `description`.
* **Параметры сортировки (`ordering`):** по полям `price`, `created_at`, `average_rating`.

* **Ответ (`200 OK`):**
```json
{
  "count": 42,
  "next": "http://127.0.0products/?page=2",
  "previous": null,
  "results": [
    {
      "id": 1,
      "title": "Корм для собак",
      "description": "Описание товара...",
      "price": "1500.00",
      "in_stock": true
    }
  ]
}
```

### 2. Создание товара
* **URL:** `products/`
* **Метод:** `POST`
* **Тело запроса (JSON):** Включает `title`, `price`, `category`, `animal` и др.

### 3. Детали товара
* **URL:** `products/<id>/`
* **Метод:** `GET`

### 4. Изменение товара
* **URL:** `products/<id>/`
* **Метод:** `PUT` / `PATCH`

### 5. Удаление товара
* **URL:** `products/<id>/`
* **Метод:** `DELETE`

### Кастомный эндпоинт: Фильтрация по массивам
* **URL:** `products/filters/`
* **Метод:** `POST`
* **Права:** `AllowAny`
* **Тело запроса (JSON):**
```json
{
  "categories": ["Сухой корм", "Лакомства"],
  "animals": ["Собаки", "Кошки"]
}
```
* **Ответ (`200 OK`):** Массив отфильтрованных товаров (без пагинации).

## Корзина (`cart/` и `cart-items/`)
Все эндпоинты корзины защищены: требуют `IsAuthenticated`.

### 1. Получить корзину текущего пользователя
Возвращает корзину текущего сессионного пользователя без необходимости передавать ID в URL.
* **URL:** `cart/`
* **Метод:** `GET`

### 2. Список элементов в корзине
Возвращает элементы корзины **только** для текущего пользователя.
* **URL:** `cart-items/`
* **Метод:** `GET`

### 3. Добавление товара в корзину
* **URL:** `cart-items/`
* **Метод:** `POST`
* **Тело запроса (JSON):**
```json
{
  "product": 1,
  "quantity": 2
}
```
* **Логика:** Если товар уже был в корзине пользователя, сервер автоматически увеличит поле `quantity` на указанное число вместо создания дубликата.

### 4. Изменение количества / Удаление из корзины
* **URL:** `cart-items/<id>/`
* **Метод:** `PUT` / `PATCH` / `DELETE`

## Опции для выпадающих списков (Form Options)
Публичные эндпоинты, возвращающие массивы (`TextChoices`) для построения селектов на фронтенде.

### 1. Опции размеров животных
* **URL:** `animal_size_options`
* **Метод:** `GET`
* **Ответ (`200 OK`):**
```json
[
  { "value": "small", "label": "Маленький" },
  { "value": "medium", "label": "Средний" }
]
```

### 2. Опции типов боксов/коробок
* **URL:** `box_type_options`
* **Метод:** `GET`
```json
[
  { "value": "cardboard", "label": "Картонная коробка" }
]
```
